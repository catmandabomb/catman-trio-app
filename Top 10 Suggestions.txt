TOP 10 SUGGESTIONS FOR CATMAN TRIO APP
========================================
Post v17.81 (ForScore-Style Charts + Pedal Support)
Generated: March 2026

---------------------------------------------------------------------
1. SONG BOUNDARY HAPTIC FEEDBACK
---------------------------------------------------------------------
When swiping to a new song (not just a new page) in Live Mode, fire a
short haptic pulse via navigator.vibrate(). Gives tactile confirmation
that you've crossed a song boundary — invaluable during a live gig when
you can't look at the screen. One line of code, huge UX win.


---------------------------------------------------------------------
2. AUTO-SCROLL / AUTO-ADVANCE MODE FOR LIVE MODE
---------------------------------------------------------------------
Add a configurable auto-scroll timer that advances pages at a set
interval (e.g., every 30s, 45s, 60s). Useful for songs where BPM is
known and chart length is predictable. Tap anywhere to pause/resume.
Could also use the BPM field to intelligently estimate page duration.


---------------------------------------------------------------------
3. SETLIST "REHEARSAL MODE" WITH SONG NOTES OVERLAY
---------------------------------------------------------------------
A variant of Live Mode that keeps a persistent translucent overlay
showing the song's notes/comments field. In performance you want clean
charts, but in rehearsal you want reminders visible. Toggle between
rehearsal and performance modes with one tap.


---------------------------------------------------------------------
4. OFFLINE PDF CACHING IN SERVICE WORKER
---------------------------------------------------------------------
Currently Drive PDFs are fetched every time. Cache the most recently
viewed charts (e.g., last 50 PDFs or last 3 setlists) in the service
worker's cache. On next open, show the cached version instantly while
checking for updates in the background. Critical for gig venues with
spotty WiFi.


---------------------------------------------------------------------
5. QUICK-JUMP SONG PICKER IN LIVE MODE
---------------------------------------------------------------------
Long-press the progress indicator (Song 3/12) to open a mini overlay
listing all songs in the setlist. Tap any song to jump directly to it.
Essential when the band calls an audible mid-set and skips ahead or
goes back.


---------------------------------------------------------------------
6. SETLIST DUPLICATION + TEMPLATES
---------------------------------------------------------------------
One-tap "Duplicate Setlist" so you can base tonight's set on last
week's. Also allow marking a setlist as a "template" that doesn't
appear in the main setlist list but shows up as a starting point when
creating new ones.


---------------------------------------------------------------------
7. DARK/STAGE-RED THEME FOR LIVE MODE
---------------------------------------------------------------------
Add an optional deep red tint for Live Mode's background and overlay
text. Red light preserves night vision on dark stages and doesn't
blind the performer. Toggle in the setlist-live header bar. Also
consider a "blackout" button that dims the screen to near-zero.


---------------------------------------------------------------------
8. PRACTICE SESSION TIMER + STATS
---------------------------------------------------------------------
When practicing a song, track time spent per song and per practice
session. Show cumulative stats on the Practice detail page — "Total
practice time: 2h 15m across 8 sessions". Helps musicians see which
songs need more work and which are well-rehearsed.


---------------------------------------------------------------------
9. MULTI-CHART SUPPORT PER SONG IN LIVE MODE
---------------------------------------------------------------------
Some songs have multiple charts (lead sheet + chord chart, or
different arrangements). Allow the primary chart star to be toggled
per-instrument or per-context. For example, "Cat's chart" vs "Trio
chart." In Live Mode, a small tab/pill lets you flip between them
for the current song without leaving the page.


---------------------------------------------------------------------
10. BATCH OPERATIONS ON SONG LIST
---------------------------------------------------------------------
When in Edit mode on the main song list, add checkboxes for multi-
select. Then offer batch actions: "Add to Setlist," "Tag as...",
"Delete selected." Saves enormous time when organizing 100+ songs
into setlists or cleaning up the repertoire.


=====================================================================
COSMETIC ITEMS (also recommended):
=====================================================================

  between songs during Live Mode carousel transitions
- Loading spinner in Live Mode should pulse/breathe rather than
  just spin (more modern feel)
- PTR indicator arrow icon (currently dead CSS with no matching HTML)
  — either add the arrow element or remove the dead CSS
